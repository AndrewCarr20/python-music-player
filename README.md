# python-music-player
cross platform music player written in python

### Components needed to run
|                |                                 |
| -------------- |:-------------------------------:|
| prog.language  | python 3.4                      |
| gui framework  | PyQt 5                          |
| media streamer | Gstreamer 1.0 (python wrapper)  |
| platform       | cross platform                  |

### How to launch
`python3 launcher.py`
